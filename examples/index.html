Shared Flags

-f Slant

Use the Slant FIGlet font.

--duration 1.6

Render each effect over 1.6 seconds.

--fps 8

Use 8 frames per second to keep GIFs small.

--seed 21

Make randomized effects reproducible.

--intensity 0.35

Use a moderate effect strength where supported.

--cell-width 7 --cell-height 14

Control exported character cell size.

--bg black

Render against a black media background.

--export PATH

Write the animation instead of only playing it in the terminal.

Other Export Formats

The gallery uses GIFs. Use the same animation flags with a different extension, or repeat --export, for WebP, MP4, and WebM.

ascii-banner -f Slant -c green --animate matrix --duration 3 --fps 15 --seed 21 --export matrix.gif "LLM-WIKI"
ascii-banner -f Slant -c green --animate matrix --duration 3 --fps 15 --seed 21 --export matrix.webp "LLM-WIKI"
ascii-banner -f Slant -c green --animate matrix --duration 3 --fps 15 --seed 21 --export matrix.mp4 "LLM-WIKI"
ascii-banner -f Slant -c green --animate reveal --export reveal.mp4 --export reveal.webm "LLM-WIKI"
